AmigaPKG

The signed package index for amipkg, the on-Amiga package manager for AmigaImager-built systems.

The index
  • packages.json — the catalog
  • packages.json.sig — detached Ed25519 signature

The signature makes the host untrusted: clients verify these bytes against a baked-in public key, so it doesn't matter where they're served from.

Software is added by pull request and reviewed + signed by a maintainer — see the repository for how to contribute. Package archives are hosted on Aminet; this index only describes and links them.

Part of AmigaImager.
